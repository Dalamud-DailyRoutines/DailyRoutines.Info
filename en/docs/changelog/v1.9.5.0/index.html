v1.9.5.0 Update Notes

Modifications on Daily Routines

Section titled “Modifications on Daily Routines”
  • An issue that the auto command completion feature didn’t work has been fixed.
  • Records of auto command completion are now stored and queued in the chat message memory.
  • The logic of auto update was adjusted. Now it adopts the same logic as Dalamud.
  • Translations of some non-critical parts in the notes are missing due to translator absence. Please feel free to seek help in the channels if there’s any confusion.

New modules

Section titled “New modules”

Auto Retrieve Identical Items From Glamour Dresser [UI_Operation]

Section titled “Auto Retrieve Identical Items From Glamour Dresser [UI_Operation]”

Recommended Module

Automatically retrieve items with the same model from Glamour Dresser. The module requires manual start.

Auto Highlight Cursor [UI_Optimization]

Section titled “Auto Highlight Cursor [UI_Optimization]”

Recommended Module

Automatically apply a highlighted background at the cursor’s position in order to emphasize it.

Better Window Minimization [General]

Section titled “Better Window Minimization [General]”

Provide new commands /pdr mini/tray as well as a button in order to make game window minimization more convenient.

Better Command Panel [UI_Optimization]

Section titled “Better Command Panel [UI_Optimization]”

Recommended Module

Provide a better Command Panel and Command Panel commands. A lock button would be introduced in order to pin Command Panel on the screen, so that the panel would adopt the same interaction logic as Hotbar and never be hidden due to ESC input, area switching or interactions. The command /quickpanel could also take a page number as the parameter for direct access of a specific page.

Auto Display Character Hitbox [UI_Optimization]

Section titled “Auto Display Character Hitbox [UI_Optimization]”

Recommended Module

Automatically draw a customizable icon at the hitbox of the character, in order to better indicate the exact position.
