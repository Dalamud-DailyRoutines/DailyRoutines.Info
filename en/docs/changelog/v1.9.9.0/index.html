v1.9.9.0 Update Notes

Modifications on Daily Routines

Section titled “Modifications on Daily Routines”
  • A game performance issue due to delayed execution of the task queue has been fixed.
  • A internal thread management which is overly complicated was removed for better performance and stability.
  • The performance while requesting data from Universalis was improved.
  • The local time generation is now more accurate.
  • Now the initialization and uninstallation are both on the main thread in order to prevent the game from crashing.
  • Now the auto command completion window will be zoomed out if there’s no suggestion available.
  • The logic of instant movement shared within Daily Routines was optimized.

Auto Redirect Action Target [Action]

Section titled “Auto Redirect Action Target [Action]”
