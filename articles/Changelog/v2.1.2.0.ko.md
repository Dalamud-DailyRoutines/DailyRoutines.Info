## 본체 변경

- 다양한 의존성 업데이트를 대량으로 통합
- 다음과 같은 편리한 텔레포 경로를 추가:
    - 시로가네 → 쿠가네
    - 검은장막 숲 동부삼림 ↔ 기라바니아 변방지대
- 플러그인 내 지역 간 이동 처리 로직에 전송망 이용권 관련 판단 로직을 추가
- 인증 서버를 사용할 수 없는 상태에서 플러그인을 로드하면 게임이 응답하지 않을 수 있는 문제를 수정
- KamiToolKit의 메모리 관리 메커니즘을 최적화

## 신규 모듈

### PvP 대상 선택 정보 서버 전송 자동 차단 [임무]

*온라인 인증*

PvP 지역에서 다른 플레이어를 타겟팅할 경우, 대상 선택 정보가 서버로 전송되는 것을 자동으로 차단합니다. 이를 통해 다른 유저들의 화면에서는 본인이 아무도 타겟팅하지 않은 것처럼 보이게 만듭니다.

## 모듈 변경

### 적대 목록 개선 [전투]

- 다음과 같은 각 항목에 설정 가능한 옵션을 추가:
    - 시전 텍스트
        - 오프셋
        - 글꼴 크기
        - 텍스트 색상
        - 테두리 색상
    - HP 텍스트
        - 오프셋
        - 글꼴 크기
        - 텍스트 색상
        - 테두리 색상
    - 어그로
        - 오프셋
        - 글꼴 크기
        - 텍스트 색상
        - 테두리 색상
    - 상태 효과
        - 오프셋
        - 배율

### 자동 돌발 임무 [스크립트]

- 모듈 로직을 완전히 재구축
- 수집형 돌발 임무 진행도가 100%에 도달했을 때, 돌발 임무 아이템을 아직 보유 중이면 즉시 제출하도록 변경
- 지역 내에 미정산된 수집형 돌발 임무가 남아있거나, 두 빛깔 보석 보상 증가 상태 효과가 적용되어 있어도 지역을 전환할 수 있었던 문제를 수정
- 마테리아 정제나 장비 수리 등의 상태에서 임무 입퇴장을 시도하던 문제를 수정
- `다중 지역` 모드에서 두 지역 사이를 반복적으로 이동하던 문제를 수정
- `오르코 파차`의 돌발 임무 `춤추는 화산` 진행 중, 비정상 지역으로 지속적으로 경로 탐색을 시도하던 문제를 수정
- 전투 상태에서 벗어나지 못하는 경우의 처리를 최적화
- 빠른 작업 플로팅 창 추가
- 모든 모드에서 콘텐츠 지역 자동 전환 지원
- 버디 초코보 태세 자동 전환 로직 추가, HP가 60% 이상일 경우 `공격 태세`로, 그 외에는 `회복 태세`로 자동 전환
- 대상 선택 로직 최적화
- 경로 탐색 로직을 통일 및 최적화
- 두 빛깔 보석 보상 증가 효과가 적용되어 있어도 지역 내에 참가 가능한 돌발 임무가 없을 경우 다른 지역으로 이동할 수 없게 되던 로직을 제거
- 최대 레벨 조율이 가능한 돌발 임무를 우선 선택하도록 변경

### 디아뎀 제도 자동 채집 [스크립트]

- 플로팅 창에 빠른 작업 버튼 표시
- 전반적인 조작 부드러움 최적화
- 용올림 사용 로직 최적화, 용올림 위에 멈춰서 발동되지 않는 문제 방지
- 플로팅 창을 연 후에는 디아뎀 제도와 창천 거리를 떠날 때까지 열린 상태 유지

- 설정 항목 `탈것 탑승 후 자동 점프` 추가, 기본적으로 비활성화
- 설정 화면의 외형 스타일 조정

### 빠른 전송 패널 [UI]

- 데이터 갱신 로직 최적화
- `/pdrtp target` 명령어가 작동하지 않는 문제를 수정
- 모듈 내부 디자인 패턴 재구축
- 돌발 임무 이름에 퍼센트 기호가 중복 표시되는 문제를 수정
- 플로팅 창 내 `제어` 패널 기능 조정
    - `순간이동 중지` 버튼 제거
    - 맵 웨이포인트 Y 좌표 수동 입력 제거
    - 좌표 오프셋 입력을 오프셋 버튼 우클릭 시 팝업 편집으로 변경
    - `안전 지역으로 복귀` 기능 추가, 잠수 상태에서만 사용 가능
    - 사용자 지정 좌표 텔레포 기능 추가, 정확한 월드 좌표를 입력하여 텔레포 가능
    - `원래 위치로 복귀` 기능을 한 번도 텔레포하지 않은 상태에서 사용할 수 있던 문제를 수정
    - 플러그인 `vnavmesh`가 활성화된 경우, `텔레포(맵 웨이포인트)` 기능이 자동으로 유효한 좌표 생성을 시도
    - `사용자 지정 텔레포 포인트` 콘텐츠 영역의 기능과 외형 스타일 조정
        - 읽기 전용 `지역` 입력 상자 제거
        - `좌표 수정` 체크박스의 명칭을 `높이 재보정`으로 변경하고, 호버 힌트 추가
        - 잘못된 아이콘 ID 입력 시 예외가 발생하는 문제를 수정
    - `블랙리스트 아이콘` 콘텐츠 영역은 아이콘을 블랙리스트에 등록한 경우에만 표시
